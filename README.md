# Лабораторная работа №2

## Браузерная история (Browser History)
Реализована система хранения истории браузера.

#Каждая запись содержит:
- URL страницы;
- дату и время посещения;
- флаг закладки.

# Реализовано
- переход назад;
- переход вперед;
- очистка истории;
- поиск по домену;
- вывод истории в виде таблицы (с выравниванием);
- вывод истории за определенный период времени.

Какая структура данных используется
- В коде используется **двусвязный список**.

Почему именно он
- Эта структура идеально подходит для истории браузера, потому что в ней каждый сайт ссылается на предыдущий и на следующий.

Главные плюсы:
- Быстрые переходы (O(1)): Кнопки «Назад» и «Вперед» работают моментально. Программе не нужно перебирать весь список с самого начала — мы просто сдвигаем указатель на один шаг назад или вперед.
- Динамическая память: Память выделяется сама: зашли на сайт — в истории появился ровно один новый элемент. Не загружает память больше необходимого.
